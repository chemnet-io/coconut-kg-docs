Querying Coconut-KG

Coconut-KG allows a variety of queries on a large number of natural products. You can query the Coconut-KG online via a SPARQL endpoint (described on this page).

Public SPARQL Endpoint

A public SPARQL endpoint for querying the Coconut-KG is at http://coconut-kg.aksw.org/sparql. OpenLink Virtuoso serves as both the back-end database SPARQL query engine and the front-end HTTP/SPARQL server with an nginx overlay primarily to cache results for each submitted query string.

NOTE: Please read the documentation and usage notes about the public SPARQL endpoint carefully before you run queries against it. Be aware that as part of our Fair Use Policy, complex queries may be subject to various restrictions and limitations.

Details of the Public SPARQL Endpoint

Endpoint type

The SPARQL Enpoint is provided by the OpenLink Software Virtuoso.

Rates and Limitations

A Fair Use Policy is in place in order to provide a stable and responsive endpoint for the community. The number of connections-per-second you can make is limited, as are result set sizes and query times, conforming to the following settings:

ResultSetMaxRows           = 10000
MaxQueryExecutionTime      =   600  (seconds)
MaxQueryCostEstimationTime =   400  (seconds)
Connection limit           =    10  (parallel connections per IP address)
