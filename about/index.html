Coconut-KG

About

The current release is a linked data version of COCONUT (COlleCtion of Open Natural ProdUcTs). COCONUT gathers data from over 50 open NP resources and contains stereochemical forms, literature, organisms that produce them, natural geographical presence and diverse pre-computed molecular properties. Our knowledge graph is called coconut-kg. For more information on the COCONUT itself, check out their documentation.

Extraction process

COCONUT provides the data as MongoDB dump. The database contains the data in several parts. The largest part is in sourceNaturalProducts. We have exported this as json. Using our ontology, we set up rules to transform. Using these rules, we transformed the data from the json file into an RDF format. The ontology can be found at http://coconut-kg.aksw.org/coco/dataset/index.html

For more information about the creation of the ontology have a look at https://github.com/chemnet-io/coconut-schema.
